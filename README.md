# PopQz
PopQz is a MERN stack quiz application that allows users to take quizzes across multiple subjects, view detailed performance analysis, and review answers. The platform also includes user authentication and a personalized dashboard for tracking progress.

## Tech Stack
- MongoDB
- Express.js
- React.js
- Node.js
- Vite

## Features
- Multiple subjects with multiple quizzes
- Quiz scoring and performance analysis
- Answer review page after quiz completion
- User authentication (login and signup)
- Personalized dashboard

## Project Structure
public – static assets  
src – React frontend  
server – backend API and database logic  

## How to Run
1. Clone the repository  
git clone https://github.com/sahanaks1011/popQZ.git

2. Install dependencies  
npm install

3. Run the application  
npm start
